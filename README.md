# CSS_fundamentals  
# CSS 11 Day Roadmap  
0. Box Model CSS  - (box-sizing, width, height, margin, padding)  
1. Display & Position  - (display, position, absolute, fixed, sticky, z-index)  
2. Flexbox CSS - (flex, justify-content, align-items, flex-direction, gap, flex-wrap)    
3. Flexbox Real Practice - (Centered Login Box)    
4. Responsive Design  - (%, vw, vh, max-width, min-height, media queries)  
5. Learn Grid CSS  - (grid, grid-template-columns, gap)  
6. Grid (Only Essentials)  - (Blog/Article Layout)  
7. Typography CSS  - (line-height, letter-spacing, font-weight, text-align)  
8. Colors UI Polish CSS  - (HEX, RGB, HSL, Contrast, border-radius)  
9. Transitions & Hover  - (transition, :hover, transform: scale/translate)  
10. Common UI Components  - (Button states, form inputs, card components, modal(basis))  
11. Cleanup + Best Practices  - (css reset, reusable classes, naming(.btn-primary, .card,))
